MyList Module
=============

USAGE
-----
Inherits from list. Has a public method that prints a sorted list

Importing
---------

>>> MyList = __import__('1-my_list').MyList

TESTS
-----

Negative Integers
-----------------
>>> my_list = MyList()
>>> my_list.append(-1)
>>> my_list.append(-4)
>>> my_list.append(-2)
>>> my_list.append(-3)
>>> my_list.append(-5)
>>> print(my_list)
[-1, -4, -2, -3, -5]
>>> my_list.print_sorted()
[-5, -4, -3, -2, -1]
>>> print(my_list)
[-1, -4, -2, -3, -5]

List with repeated values
-------------------------

>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(2)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 2, 5]
>>> my_list.print_sorted()
[1, 2, 2, 4, 5]
>>> print(my_list)
[1, 4, 2, 2, 5]

Test inheritance
----------------

>>> issubclass(MyList, list)
True

Test empty list
---------------
>>> my_list = MyList()
>>> print(my_list)
[]
>>> my_list.print_sorted()
[]
